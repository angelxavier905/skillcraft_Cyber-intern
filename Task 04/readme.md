# Keylogger Tool 

This is a basic keylogger program built using Python.
It records the keys pressed on the keyboard and saves them into a file.

## What this program does

- Captures keystrokes from the keyboard  
- Logs each key pressed  
- Saves the data into a text file  

## How to run

1. Install required library:
   pip install pynput

2. Run the Python file:
   python keylogger.py

3. Type anything on your keyboard  

4. Press ESC to stop logging  

## Output

- The recorded keys are saved in:
  keylog.txt  

Example:
hello123[Key.space]test

## Important Note ⚠️

This project is created for educational purposes only.
It is meant to demonstrate how keystroke logging works in cybersecurity.  
Do not use this program on others' systems without permission.
