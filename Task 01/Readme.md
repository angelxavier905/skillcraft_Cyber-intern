# Caesar Cipher Tool 

This is a simple Python program that performs encryption and decryption using the Caesar Cipher technique.
I created this as part of my cybersecurity internship to understand how basic encryption works.

## What this program does

- Encrypts a message using a shift value  
- Decrypts an encrypted message back to original  
- Works with uppercase and lowercase letters  
- Keeps spaces and symbols unchanged  

## How to run

1. Open the file in Python
2. Run the program
3. Choose:
   - 1 for Encrypt
   - 2 for Decrypt
4. Enter your message and shift value
